Package: Controle
Esta classe exibirá a página de definições, com ela será possivel configura tanto a catraca virtual quanto a catraca física.

Nesta página serão definidos vários parêmetros para o correto funcionameto das catracas, como:

1 - Cadastro, edição e configuração de Unidades Academicas;

2 - Cadastro, edição e configuração de Catracas Virtuais;

3 - Cadastros, edição e configuração de Turnos;

4 - Cadastro, edição e configuração de Tipos de Usuário e seus valores;

5 - Cadastro e edição de mesnagens para catraca física;

6 - Definnição de valores de custo das refeições por Unidade Acadêmica.

telaCatracas() : void

Esta função gera a tela de configuração das catracas tanto vitual quanto fisica.

Nela será possível cadastrar novas catracas virtuais e configura-las, podendo definir a sua Unidade Acadêmica, assim como Habilitar ou Desabilitar o módulo financeiro.

A catraca física possui um auto cadastro, porém pode ser configurada nesta tela, pode ser definido:

1 - sentido do giro;

2 - o tempo de giro;

3 - ip da máquina;

4 - a interface de rede (Cabeada ou Sem-fio);

5 - Habilitar ou Desabilitar o módulo financeiro.

Também será verificado a dualidade de nomes das catracas.

telaConfiguracaoDeTurnos() : void

Esta tela gera a tela de configuração de turnos, nela é possível cadastrar, editar, excluir e definir os horários de inicio e fim de funcionamento de cada turno

telaConfiguracaoUnidadeAcademica() : void

Esta função gera a tela de configuração da Unidade Acadêmica, onde será possível cadastrar, editar e icluir novos turnos a cada unidade acadêmica.

Ela vefifica a dualidade de nomes das Unidades.

telaDeCustos() : void

Esta função gera a tela de configuração dos valores de custo das refeições e cartão.

É possivel definir o valor de custo de cada unidade.

telaDefinicoes() : void

Esta função gerência as tabs com as telas de configurações.

cada tab realiza a chamada da função referente a sua tela.

telaMensagem() : void

Esta função Gera a tela de cadstro e edição das mensagens da catraca física.

Cada catraca terá as suas próprias mensagens.

telaTiposDeUsuarios() : void

Esta função gera a tela de configuração de Tipos de Usuário.

Nela é possível Cadastrar e editar cada tipo, definido o valor que será cobrado em cada refeição.
