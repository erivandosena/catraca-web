CartaoDAO

Extends \DAO

Nesta classe são realizadas as conexões com o Banco de Dados, com referência ao tratamento de dados relativos a entidade Cartões.

Nela é realizada Consulta, Inserção, Atualização e Exclusão dos cartões.
