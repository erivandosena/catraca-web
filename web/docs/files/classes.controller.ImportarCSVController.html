Package: Controle
Esta classe é utilizada para a importação dos quandos quando ocorrer a falta de conexão com o servidor.

Através do Bloco de Notas criado o arquivo .csv contendo os números dos cartões que utilizaram o RU.

O valor da refeição será creditado de cada cartão.

Methods

importar() : void

Importa o arquivo para processar os dados e debitar os valores de cada cartão.
